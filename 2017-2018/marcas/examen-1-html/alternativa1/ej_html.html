Los lenguajes de marcas

Versiones

A día de hoy se puede decir que conviven tres grandes familias de lenguajes de marcas para la web:
  • HTML 4.1
  • XHTML
  • HTML 5
Algunos autores definen así los siguientes términos
HTML
Lenguaje de marcas para la Web
XML
Lenguaje de marcas para intercambio de datos
