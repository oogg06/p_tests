\pregunta{Elaborar una DTD que permita validar un fichero XML que se atenga a las reglas siguientes:}{4}
\begin{itemize}
\item{    El elemento raíz se llama documentos.}
\item{    Dentro de él puede haber uno o más elementos documento.}
\item{    Un documento debe empezar con un elemento creador, que siempre es una cadena de texto o empezar
por un elemento vacío llamado {\tt creadordesconocido}}
\item{    Un documento siempre debe llevar una fecha de creación. La fecha debe llevar siempre un atributo
llamado {\tt formato} que lleva dentro una cadena de texto.}
\item{    Un documento siempre debe llevar un elemento llamado {\tt uso}. El uso siempre será una
de estas tres cadenas de texto: {\tt Privado.} {\tt Público} o {\tt Desconocido}     }
\item{    Todo documento debe llevar un elemento {\tt paginas} que debe llevar dentro un número entero positivo.}
\end{itemize}

\end{itemize}
