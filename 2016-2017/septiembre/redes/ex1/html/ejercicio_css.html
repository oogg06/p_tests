La importancia de CSS

El lenguaje CSS se creó por diferentes motivos:

  1. Se necesitaba separar las marcas de su estética.
  2. Era importante tener en un solo fichero todo el estilo de una web.
  3. Se deseaban poder incluir efectos modernos en la Web.

La evolución de las versiones fue la siguiente:

1994Hakom Wium Lie sugiere CSS
1995Se hace la propuesta oficial
1997Se forman 3 grupos de trabajo

Separación de texto y presentación.

Siempre que CSS y HTML intervienen ocurre que el diseñador
puede mantener en ficheros distintos el contenido y el estilo del sitio.Gracias a esto:

  • Cambiando un solo fichero cambia todo el sitio web.
  • Diseñador y programador pueden trabajar por separado
  • Usando JS podemos ofrecer varios estilos al usuario.
