\pregunta{Dado el archivo XML que se puede encontrar al final, 
crear una hoja de estilo XSLT que genere el HTML necesario para crear una tabla en la que aparezcan los portatiles cuya RAM sea menor de 4GB teniendo en cada fila una celda para el peso y otra para la RAM
}{3}
