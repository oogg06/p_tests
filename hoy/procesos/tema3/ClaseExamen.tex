\pregunta{Un servicio de validación funciona en base al siguiente protocolo:

\begin{enumerate}
\item{El cliente envia el mensaje HOLA.}
\item{El cliente espera que el servidor le envíe 2 lineas. Una contiene el texto BIENVENIDO y la otra contiene un número que actuará como código.}
\item{El cliente tiene que multiplicar por 2 el número que recibió del servidor.}
\item{El cliente envía al servidor dos líneas. Una contiene el texto CODIGO y la otra contiene el código de validación.}
\item{El cliente espera que le envíen una línea que contendrá un mensaje que puede ser de éxito o de fallo.}
\end{enumerate}

Crea un cliente Java que implemente este protocolo. Se valorará la calidad del código de la solución.}{3.5}
